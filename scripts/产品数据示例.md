# 飞书表格产品数据示例

根据你的表格结构，以下是建议添加的产品数据。你可以直接复制到飞书表格中。

## 表格结构说明

你的表格列：
- **分类**：产品分类（如：虾类、鱿鱼类、鱼类、贝类、干海鲜）
- **产品名称**：具体产品名称
- **图片**：产品图片的 URL 链接（飞书会自动转换为链接对象）
- **原材料**：产品原材料说明
- **尺寸**：产品规格尺寸
- **最小订单**：最小订购量
- **包装**：包装方式
- **付款方式**：付款方式
- **供应商**：供应商名称

## 使用说明

1. **复制数据到飞书表格**：
   - 从第3行开始（第2行是表头），逐行粘贴上述数据
   - 图片列可以直接粘贴 URL，飞书会自动识别为链接

2. **图片 URL 说明**：
   - 示例中的图片 URL 是占位符，你需要替换为实际的图片链接
   - 或者直接在飞书表格中上传图片，飞书会自动生成链接

3. **数据格式**：
   - 确保每列的数据格式正确
   - 分类名称要保持一致（用于自动分组）
   - 供应商名称建议统一为：`MARINE FOOD PRODUCTS CO., LTD.`

4. **测试**：
   - 添加数据后，运行 `node scripts/sync-feishu.js` 测试
   - 检查生成的 `data/products.yaml` 是否正确

## 注意事项

- 图片 URL 如果是完整链接（http:// 或 https:// 开头），会直接使用
- 如果图片字段为空，会使用默认图片 `images/product-default.svg`
- 确保"产品名称"字段不为空，否则该行会被跳过
- 确保"分类"字段不为空，否则该行会被跳过
